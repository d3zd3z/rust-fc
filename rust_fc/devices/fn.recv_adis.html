pub fn recv_adis(message_buffer: &[u8]) -> ADIS

Read an ADIS message from raw bytes.

Unrwap a byte array (assuming network endianess) into fields for the ADIS Data type. This will also do the conversion from ADC counts to proper (SI) Units.

Parameters

  • message_buffer: A buffer of bytes at least as long as the ADIS message (SIZE_OF_ADIS).
